### Installation
```bash
$ [sudo] pip install mac-app-pid
```

```bash
$ [sudo] npm i -g mac-app-pid
```

#### Features
+   accepts app name, bundleID, path

#### Examples
```bash
$ app-pid "itunes"
<pid>
$ app-pid "com.apple.iTunes"
<pid>
$ app-pid "/Applications/iTunes.app"
<pid>
```
